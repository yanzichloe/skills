# Forbidden Words

## 1. 禁用词使用原则

以下词汇和表达在金融营销、产品介绍、投顾服务、AI 回复、交易流程中应默认禁用或高度谨慎。

判断标准：

```text
是否承诺收益？
是否暗示保本？
是否夸大确定性？
是否诱导冲动交易？
是否弱化风险？
是否制造不适当的财富焦虑？
```

## 2. 收益承诺类禁用词

```text
稳赚
稳赚不赔
保赚
必赚
躺赚
稳赚收益
锁定收益
固定收益
收益保证
保证赚钱
```

替代表达：

```text
历史表现
收益表现
存在波动风险
不代表未来收益
```

## 3. 保本 / 低风险误导类禁用词

```text
保本
本金无忧
零风险
无风险
低风险高收益
安全收益
稳赚型产品
放心买
闭眼买
```

替代表达：

```text
风险等级相对较低
波动相对较小
仍存在本金损失风险
请结合风险承受能力判断
```

## 4. 未来预测类禁用词

```text
马上上涨
必然反弹
确定底部
抄底机会
未来收益可期
预计收益率
未来一年收益 X%
下一个爆款
```

替代表达：

```text
可以关注
历史数据显示
从当前情况看
仍存在不确定性
```

## 5. 过度紧迫类禁用词

```text
最后机会
错过就亏
立即抢购
马上上车
限时疯抢
再不上车就晚了
倒计时赚钱机会
```

替代表达：

```text
阶段性开放
名额以页面展示为准
可在了解风险后决定是否参与
```

## 6. 排名绝对化禁用词

```text
第一
最强
最佳
顶级
无敌
行业唯一
全网最稳
收益最高
```

替代表达：

```text
根据 XX 数据源，截至 XX 日期，在 XX 统计口径下排名 XX
```

## 7. 私募 / 高净值高风险词

```text
内部机会
私募稳赚
富人专属赚钱通道
财富捷径
高净值闭眼买
稳赚私募
专属暴富机会
```

替代表达：

```text
面向满足适当性要求的用户开放
适合进一步了解复杂资产配置服务
需完成合格投资者认证后查看
```

## 8. AI 场景禁用词

```text
我保证
我预测
我确定
最佳买点
确定方案
自动赚钱
跟我买
我已经替你判断
```

替代表达：

```text
我可以帮你分析
基于当前信息
仅供辅助理解
仍需你结合自身情况判断
```

## 9. 文案替换表

| 高风险表达 | 替代表达 |
|---|---|
| 稳赚不赔 | 历史表现不代表未来，投资存在风险 |
| 马上抄底 | 可以关注当前估值和风险变化 |
| 锁定收益 | 了解收益来源和波动风险 |
| 最后上车 | 阶段性开放，参与前请阅读规则 |
| 内部机会 | 面向符合条件用户开放的服务说明 |
| 低风险高收益 | 风险收益特征需结合产品文件理解 |
| AI 判断一定涨 | AI 分析仅供辅助理解，不构成收益保证 |

## 10. 检查清单

- [ ] 是否出现“稳、保、必、锁定、无风险”？
- [ ] 是否出现绝对排名或绝对收益？
- [ ] 是否把不确定性写成确定性？
- [ ] 是否制造过度稀缺或焦虑？
- [ ] 是否存在私募公开推介风险？
- [ ] 是否存在 AI 过度承诺？

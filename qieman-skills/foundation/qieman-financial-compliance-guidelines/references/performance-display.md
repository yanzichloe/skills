# Performance Display

## 1. 适用范围

本文件用于规范收益、业绩、回撤、分红、排行榜、图表等展示方式。

适用页面：

- 基金详情页
- 投顾策略页
- 组合页
- 财富报告
- 账户诊断报告
- 营销 H5
- 高净值宣传页
- 私域海报
- AI 小顾分析卡片

## 2. 核心原则

业绩展示必须满足：

```text
数据真实、区间明确、口径清楚、来源可追溯、风险并列表达
```

## 3. 必须补充的信息

当展示收益或业绩时，应尽量同时展示：

- 数据区间
- 计算口径
- 数据来源
- 是否扣费
- 是否为历史数据 / 模拟测算 / 假设情景
- 比较基准
- 风险提示

## 4. 历史业绩规范

### 可以

```text
近一年收益率：X%
数据区间：YYYY-MM-DD 至 YYYY-MM-DD
数据来源：基金定期报告 / 平台统计 / 第三方数据源
过往业绩不代表未来表现。
```

### 避免

```text
未来一年预计收益 X%
长期稳赚 X%
年化收益可达 X%
历史收益这么高，未来也值得期待
```

## 5. 测算 / 假设收益规范

涉及测算时必须明确：

- 假设条件
- 测算方法
- 不构成未来收益承诺
- 实际收益可能不同

推荐模板：

```text
以下为基于假设条件的测算示例，不代表实际收益，也不构成收益承诺。实际结果受市场波动、产品费用、交易确认时间等因素影响。
```

## 6. 排名 / TOP 表达规范

可用前提：

- 数据来源明确
- 统计口径明确
- 统计区间明确
- 排名对象可比
- 不用永久性绝对表达

推荐：

```text
根据 XX 数据源，截至 YYYY-MM-DD，在 XX 类型产品中排名第 X。
```

避免：

```text
全市场第一
最强策略
永远领先
行业最佳
闭眼买
```

## 7. 图表规范

金融图表必须：

- 坐标轴清晰
- 标明时间区间
- 标明数据来源
- 收益与回撤尽量并列展示
- 不通过裁剪坐标轴夸大收益
- 不通过颜色制造过度乐观情绪
- 不隐藏最大回撤、波动率等重要风险信息

## 8. 分红 / 现金流展示规范

推荐：

```text
历史分红情况仅供参考，不代表未来分红水平。基金分红受基金合同、收益分配政策和市场情况影响。
```

避免：

```text
每月固定到账
稳定领钱无风险
年年分红不用愁
```

## 9. 收益展示检查清单

- [ ] 是否标明历史区间？
- [ ] 是否标明数据来源？
- [ ] 是否说明过往不代表未来？
- [ ] 是否避免预期收益率表达？
- [ ] 是否避免选择性展示最优区间？
- [ ] 是否补充风险指标或风险提示？
- [ ] 是否避免图表视觉夸大？

## 10. 监管依据摘要

公开募集基金宣传推介材料登载过往业绩时，应满足基金合同生效时间、业绩区间、比较基准、数据来源和风险提示等要求；展示过往业绩时应声明过往业绩不预示未来表现，不得承诺未来收益。

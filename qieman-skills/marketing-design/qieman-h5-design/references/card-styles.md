# 且慢营销 H5 卡片样式规范

## Purpose
本文件用于定义且慢营销 H5 中卡片类模块的设计规则，帮助在活动页、专题页、品牌传播页、年度账单页等场景中，统一信息卡、数据卡、引导卡、活动卡和对比卡的视觉表现与结构逻辑。

本规范重点解决以下问题：

- 不同类型卡片的适用场景
- 卡片的层级、信息密度与视觉节奏
- 圆角、阴影、边框、留白等基础样式
- 数据类与营销类卡片的差异化表达
- H5 长页中多卡片组合时的统一性与可读性

---

## When to read
在以下情况应优先读取本文件：

- 需要设计营销 H5 页面中的卡片模块
- 需要组织摘要信息、数据亮点、引导入口或活动卖点
- 需要统一不同页面中卡片组件的视觉风格
- 需要输出适用于 Figma、即梦、Lovart、HTML Demo 的卡片描述
- 需要在 H5 长页中平衡「信息清晰」与「营销表现力」

---

## Key rules

### 1. 卡片优先承担「信息组织」而不是装饰
卡片的核心作用是帮助用户快速理解信息层级、重点内容和操作路径，不应为了视觉热闹而破坏可读性。

### 2. 一个卡片只表达一个核心主题
单卡片内应尽量只承载一个主要信息目标，例如：

- 一个数据结论
- 一个用户洞察
- 一个活动权益
- 一个操作引导
- 一个配置说明

不要在一个卡片中混合过多并列信息。

### 3. 卡片内部层级必须清晰
通常遵循以下顺序：

1. 标题 / 标签
2. 核心数值 / 核心卖点
3. 补充说明
4. 辅助图形 / 图标 / CTA

### 4. 卡片视觉应保持轻盈、可信、克制
且慢营销 H5 中的卡片需要体现：

- 专业感
- 金融服务的可信度
- 轻盈留白
- 信息秩序
- 可复用性

### 5. 多卡片组合时保持统一
同一页面中的卡片应尽量统一以下维度：

- 圆角体系
- 阴影强度
- 内边距规则
- 标题与正文层级
- 边框与背景处理方式

---

## Recommended structure

### 单卡片推荐结构
```text
标题 / 标签
核心信息
补充说明
辅助元素（可选）
CTA（可选）
```

单卡片内信息自上而下排列，标题与核心信息优先，补充说明和 CTA 次之；辅助图形或图标不抢主信息。

### 双列卡片区推荐结构
```text
分区标题（蓝竖条 + 19pt）
┌─────────────┬─────────────┐
│  卡片 A     │  卡片 B     │
│  核心信息   │  核心信息   │
│  补充说明   │  补充说明   │
└─────────────┴─────────────┘
```

适用场景：两项并列数据、两种方案对比、两个入口引导。两卡等高或按内容自然高度，间距 12px～16px；移动端可改为上下堆叠。

### 多卡片连续区推荐结构
```text
分区标题
卡片 1（如年度亮点摘要）
卡片 2（如数据对比）
卡片 3（如引导 / 权益）
…
```

卡片之间间距 16px；同一分区内卡片风格统一（圆角、内边距、背景处理一致）；可配合顶条标题卡、数据卡、引导卡等类型组合使用。

## Base style guidelines

| 属性       | 取值 |
|------------|------|
| 背景       | 常规卡片 `#FFFFFF`；页面底 `#F9FAFB` |
| 圆角       | 8px / 12px |
| 内边距     | 12px～16px |
| 模块间距   | 16px |
| 边框       | 默认无；摘要/高亮卡可用 `1px solid #9CCBF8` 或 `#FFDD99` |
| 阴影       | 无或极轻；以留白区分层次，不依赖重阴影 |

分区标题：蓝竖条 4px×20px、`#1B88EE`，与标题文字间距 8px；标题 19pt 加粗、`#333333`。

主 CTA：背景 `#1B88EE`，文字 `#FFFFFF`，圆角 8px 或胶囊形 9999px，内边距 12px 24px；无边框、无投影。

## Card types
且慢营销 H5 中常用卡片类型包括：信息摘要卡、数据卡、引导卡、活动权益卡、对比卡、内容组合卡。以下分节说明各类型的适用场景、视觉要点与推荐结构。

---

## 1. 信息摘要卡
**适用场景**：年度亮点、关键结论、一句话总结、数据口径说明。

**视觉要点**：
- 底色可为浅蓝 `#DBEBFF` 或浅黄 `#FFF7E5`，边框 `1px solid #9CCBF8` 或 `#FFDD99`
- 圆角 12px，内边距 16px
- 结论（大字）> 数据（对齐）> 解释（小字）

**推荐结构**：标题/标签 → 核心结论或 2×2 数据格 → 补充说明（可选）→ 数据来源/统计周期（12pt #999999）。

---

## 2. 数据卡
**适用场景**：年度收益率、资产峰值、配置占比、定投次数、收益排名等数据展示。

**视觉要点**：
- 背景 `#FFFFFF`，圆角 12px
- 大数字 16～24pt 加粗，小标签 12～14pt、`#999999`
- 正收益用 `#FF6633`，负收益用 `#07AD8F`
- 若含柱状图/环形图：深蓝 `#125EA5`、金 `#FFCC66`、品牌蓝 `#1B88EE` 等，与配色规范一致
- **必附**数据来源与统计周期，12pt、`#999999`

**推荐结构**：卡片标题 → 主指标或图表+图例 → 补充说明 → 数据来源与统计周期。

---

## 3. 引导卡
**适用场景**：下一步操作引导、功能入口、生成海报、分享账单等。

**视觉要点**：
- 背景 `#FFFFFF` 或轻底 `rgba(27,136,238,0.05)`，圆角 12px
- 主 CTA 品牌蓝 `#1B88EE`，次 CTA 文字蓝或轻底
- 标题与按钮层级清晰，一卡一主行动

**推荐结构**：标题/一句话说明 → 主 CTA → 次 CTA 或辅助说明（可选）。

---

## 4. 活动权益卡
**适用场景**：活动规则、权益清单、奖励说明、参与条件。

**视觉要点**：
- 背景 `#FFFFFF`，圆角 12px；顶条可选浅蓝/浅橙/浅紫（`#DBEBFF` / `#FDDBD3` / `#E7DBFF`）
- 标题 16pt～19pt 加粗，正文 16pt
- 列表项可配小图标，颜色 `#333333` / `#999999` / `#1B88EE`

**推荐结构**：顶条+标题 → 权益条目（图标+标题+描述）→ 规则或时间（小字）→ CTA（可选）。

---

## 5. 对比卡
**适用场景**：用户对比（如 VS 双柱）、方案对比、且慢 vs 其他。

**视觉要点**：
- 左灰右粉或左蓝右橙等对比色，与《且慢营销配色规范》一致
- 对比结论句含倍数或情绪词（如「超越 90% 用户」）
- 数据格可白底，柱状图深蓝 `#125EA5` 与金 `#FFCC66` 交替

**推荐结构**：对比标题 → 双列/双柱数据或图示 → 结论句 → 数据来源（可选）。

---

## 6. 内容组合卡
**适用场景**：顶条标题+正文+图位、分区说明+多块信息、时间线+描述。

**视觉要点**：
- 背景 `#FFFFFF`，圆角 12px，内边距 16px
- 顶条可选彩色（浅蓝/浅橙/浅紫），标题与正文层级分明
- 时间线：竖线 `#DBEBFF`，节点圆点 `#1B88EE`，与卡片白底区分

**推荐结构**：顶条+标题（或分区蓝竖条+标题）→ 正文/时间线/列表 → 底部图位或说明（可选）。

---

## Dos
- 一个卡片一个主题，信息层级清晰
- 统一圆角、内边距、模块间距
- 数据类卡片必附数据来源与统计周期
- 主 CTA 使用品牌蓝 `#1B88EE`，胶囊形或 8px 圆角
- 多卡片同屏时风格统一
- 摘要卡、数据卡用留白和色块区分层次，避免重阴影

---

## Don'ts
- 不要单卡内堆砌过多并列主题
- 不要用强阴影、高饱和撞色破坏金融感
- 不要省略数据口径或统计周期（数据卡、年度账单）
- 不要在同一页面混用多套圆角或内边距体系
- 不要用装饰抢过核心信息

---

## Examples

**示例 1：信息摘要卡（年度亮点）**  
分区标题「年度亮点」+ 浅蓝底卡片（`#DBEBFF`，边框 `#9CCBF8`），内为 2×2 白底数据格：年度累计投入、年度收益率（正收益橙红）、资产峰值、投资天数；数值 22pt 加粗，标签 12pt #999999；无 CTA。

**示例 2：数据卡（资产配置）**  
分区标题「资产配置」+ 白底卡片，内为环形图（品牌蓝/金/灰三色）+ 图例（股票型/债券型/现金货基占比）+ 底部一句说明；文末 12pt #999999「数据来源：且慢组合持仓，统计周期 2025-01-01 至 2025-12-31」。

**示例 3：引导卡（结尾 CTA）**  
标题「生成你的年度海报」，副标题「一键生成可分享的年度账单长图」；主按钮「生成年度海报」品牌蓝胶囊，次按钮「分享账单」轻底+蓝字；背景浅蓝 `#F5F9FF`，居中排版。

**示例 4：活动权益卡**  
顶条浅蓝 `#DBEBFF`，标题「活动规则」；正文列表：时间、对象、门槛、发放方式；底部小字免责；可选主 CTA「立即参与」。

**示例 5：对比卡（用户对比）**  
标题「你的投资画像」；左列浅灰柱+气泡，右列浅粉柱+气泡，中间「VS」蓝红区分；底部结论「超越 75% 用户」；数据来源 12pt #999999。

**示例 6：内容组合卡（时间线）**  
分区标题「年度时间线」+ 白底卡片；左侧竖线 `#DBEBFF`，节点圆点 `#1B88EE`；每项为月份（12pt #999999）+ 描述（15pt #333）；5 个节点纵向排列，间距 20px。

# Marketing H5 References

本目录为 **`marketing-h5`** 主 Skill 的参考规范文档集合，供按需查阅与引用，不单独作为 Skill 使用。

## Files

| 文件 | 用途 |
|------|------|
| `card-styles_zh.md` | 卡片类型、层级、圆角与留白；信息摘要卡、数据卡、引导卡、活动权益卡、对比卡、内容组合卡 |
| `color-guidelines_zh.md` | 主色 / 辅助色 / 背景色 / 文本色 / 状态色；场景色板；主 CTA 固定品牌蓝 |
| `header-hero-guidelines_zh.md` | 首屏 Hero 布局、标题层级、主视觉素材、CTA 位置；年度账单 / 活动 / AI 专题推荐结构 |
| `marketing-design-spec_zh.md` | 页面整体结构、模块节奏、信息优先级、转化路径、品牌一致性 |

## How to use

- 设计首屏 / Hero、主视觉、标题与 CTA → 读 `header-hero-guidelines_zh.md`
- 设计卡片模块、数据卡、摘要卡、引导卡 → 读 `card-styles_zh.md`
- 确定配色、背景、按钮、场景色板 → 读 `color-guidelines_zh.md`
- 规划整体结构、模块顺序、转化路径 → 读 `marketing-design-spec_zh.md`

## Combination guidance

- 做完整 H5 规划或 HTML Demo：先读 `marketing-design-spec_zh.md` 定骨架，再按模块读取 `header-hero-guidelines_zh.md`（首屏）、`card-styles_zh.md`（内容卡）、`color-guidelines_zh.md`（色值）
- 做单模块设计：按模块类型读取对应 reference；涉及配色时配合 `color-guidelines_zh.md`
- 多文件共用时保持主 CTA、品牌蓝、圆角 / 间距与卡片规范一致

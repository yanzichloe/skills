# Private Fund Rules

## 1. 适用范围

本文件用于约束且慢私募基金、高净值、300 万+、专属圈层、私募专区等相关页面和营销表达。

适用场景：

- 私募专区入口
- 私募产品列表 / 详情
- 高净值用户营销海报
- 300 万+ 用户筛选活动
- 顾问 1V1 私募沟通
- 私募相关 H5 / Banner / 社群文案

## 2. 核心原则

私募相关内容必须优先遵循：

```text
特定对象、合格投资者、适当性匹配、风险充分、不得公开推介
```

## 3. 合格投资者表达

可用于页面说明：

```text
私募基金仅面向符合条件的合格投资者开放，需完成身份验证、风险测评和适当性匹配后查看。
```

```text
相关产品具有较高风险和流动性限制，请确认自身风险识别能力和风险承受能力。
```

## 4. 300 万+ 表达规范

### 可以

```text
面向满足相应资产证明和适当性要求的用户开放。
```

```text
需完成合格投资者认证后查看具体产品信息。
```

```text
300 万+ 是合格投资者识别过程中的重要参考条件之一，最终以平台认证和适当性匹配结果为准。
```

### 避免

```text
300 万以上才能进的赚钱圈
富人专属收益机会
内部私募稳赚机会
高净值用户闭眼买
专属暴富通道
```

## 5. 私募营销限制

私募内容不得面向不特定公众进行具体产品推介。

设计上应注意：

- 不在公开海报中展示具体私募产品名称、业绩、购买入口。
- 不使用“私募收益机会”“内部标的”等刺激性文案。
- 不用朋友圈裂变、抽奖、拉人奖励等方式推广私募产品。
- 不在未认证用户页面暴露具体私募产品详情。
- 不以“限时抢购”“最后席位”作为主要营销主题。

## 6. 私募详情页风险提示

推荐模板：

```text
私募基金具有较高风险，存在本金损失、流动性受限、信息披露频率较低等风险。投资前请仔细阅读基金合同、风险揭示书等文件，并确认产品风险等级与你的风险承受能力相匹配。
```

## 7. 高净值圈层表达

高净值用户场景应表达“匹配复杂资产需求”，而不是“制造阶层感”。

### 推荐

```text
为更复杂的资产配置需求，提供更深入的顾问沟通。
```

```text
与同频投资者交流长期配置思路。
```

### 避免

```text
财富阶层专属
普通人进不来
有钱人的赚钱通道
资产越多越该买
```

## 8. 私募页面检查清单

- [ ] 是否限制为认证 / 合格投资者可见？
- [ ] 是否避免公开展示具体私募产品收益？
- [ ] 是否明确高风险和流动性风险？
- [ ] 是否避免保本、定期付息、高收益诱导？
- [ ] 是否避免不特定公众传播？
- [ ] 是否避免用财富优越感刺激转化？
- [ ] 是否明确最终以适当性匹配结果为准？

## 9. 参考口径摘要

私募基金投资者通常需要满足单只私募基金投资额不低于一定门槛，并具备相应风险识别和承受能力；个人金融资产、收入等条件常用于合格投资者判断。私募产品宣传应避免公开推介和保本保收益等违规表达。

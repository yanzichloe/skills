# Forbidden Style / 禁用设计规范

## 1. 总原则

且慢品牌不应通过强刺激、强营销、强炫富或强交易暗示来获得用户注意。

禁用方向的核心判断标准：

> 是否会让用户产生冲动投资、收益确定、财富炫耀、金融焦虑或不专业的感受。

## 2. 视觉禁用项

### 2.1 金融暴富符号

避免使用：

- 钞票堆；
- 金条；
- 大量金币；
- 暴涨箭头；
- 一夜暴富视觉；
- 彩票式财富暗示。

### 2.2 传统金融陈词滥调

避免使用：

- 城市 CBD 天际线滥用；
- 夸张商务握手；
- 西装精英强摆拍；
- 冷硬金属背景；
- 复杂交易大厅；
- 密集 K 线屏幕。

### 2.3 强刺激色彩

避免使用：

- 大面积纯黑；
- 大面积强红；
- 大面积绿色；
- 高饱和红绿对比；
- 强金色铺满；
- 夜店感霓虹科技色。

### 2.4 低幼化表达

避免使用：

- 过度卡通；
- 玩具感过强；
- 表情过度夸张；
- 幼儿化比例；
- 过分可爱化金融场景。

### 2.5 信息干扰

避免：

- 屏幕内出现不可控错误文字；
- 图表读数被装饰遮挡；
- 风险提示隐藏在角落；
- 重要数字缺少口径；
- 大量标签堆叠导致用户无法判断重点。

## 3. 文案禁用项

禁止或慎用：

```text
稳赚
保本高收益
闭眼买
马上抄底
错过再等一年
唯一机会
财富捷径
无风险收益
稳赚不赔
确定上涨
必赚
翻倍
躺赚
```

## 4. 交互禁用项

避免：

- 默认勾选高风险交易协议；
- 用关闭困难迫使用户点击主按钮；
- 弹窗连续阻断用户离开；
- 将风险提示折叠到不可见；
- 高风险操作按钮过于醒目且无二次确认；
- 用红包、倒计时等强促销元素包装投资决策。

## 5. 营销禁用倾向

避免：

- 把投资服务包装成“确定性机会”；
- 用极端收益案例刺激用户；
- 只讲收益，不讲风险；
- 过度制造稀缺；
- 把高净值圈层表达成炫富或身份优越；
- 把 AI 建议包装成最终投资结论。

## 6. AI 生成图片特别注意

生成图片时必须避免：

- 错误文字；
- 乱码文字；
- 绿色大面积出现；
- 城市背景过重；
- 过多金币；
- 金融屏幕密集数据；
- 人物低幼或表情夸张；
- IP 形象比例失控；
- 违反品牌主色的画面。

## 7. 快速检查表

输出前检查：

- 有没有收益承诺或稳赚暗示？
- 有没有刺激用户马上交易？
- 有没有过多钱币、金条、豪宅、豪车？
- 有没有大面积绿色、纯黑、强红、强金？
- 有没有传统金融假商务感？
- 有没有低幼化？
- 风险提示是否清晰？
- 画面是否还保留且慢的清爽、专业、温和感？

# Illustration Style / 插画图形风格规范

## 1. 插画定位

且慢插画用于降低金融理解门槛，表达陪伴感、专业感和轻科技感。

插画不应成为装饰堆叠，而应帮助解释业务、安抚情绪或建立品牌记忆。

## 2. 两条主线

### 2.1 产品 UI 内插画

风格：

```text
轻量
柔和
功能解释型
安抚型
低复杂度
```

适用：

- 空状态；
- 引导页；
- 弹窗说明；
- 功能入口；
- 账户诊断；
- 风险提示。

### 2.2 营销 / 品牌插画

风格：

```text
情绪化
场景化
温暖陪伴
自然意象
轻科技质感
```

适用：

- H5；
- 海报；
- Banner；
- 节日活动；
- AI 小顾宣传；
- 高净值权益。

## 3. 类型规范

| 类型 | 风格要求 |
|---|---|
| 功能图标 | 线面结合、圆角、浅蓝底、少量暖色点缀 |
| 空状态插画 | 简单、安抚、不夸张，给出下一步行动 |
| 金融场景插画 | 人物友好，场景真实，避免金融符号堆砌 |
| AI 小顾插画 | 更亲和、更智能，但不能低幼化 |
| 3D 插画 | 柔光、圆润、品牌蓝、轻科技质感 |
| 高净值插画 | 几何、金属、蓝金、低饱和、克制高级 |

## 4. 常用元素

- 小船、海面、星空、山丘、花朵；
- 账本、卡片、图表、资产配置模块；
- 顾问、用户、家庭、规划场景；
- AI 节点、透明卡片、数据流、轻科技图形；
- 徽章、金属边框、几何资产结构。

## 5. 图标规范

金融功能图标建议：

- 圆角几何；
- 线面结合；
- 主色品牌蓝；
- 暖色点缀不超过 20%；
- 避免绿色；
- 背景可用浅蓝圆形或圆角矩形；
- 金融符号使用要简洁，不堆叠。

## 6. AI 小顾相关视觉

AI 小顾视觉应表达：

- 可信；
- 聪明；
- 亲和；
- 专业；
- 有陪伴感。

避免：

- 过度低幼；
- 玩具感过强；
- 夸张表情；
- 过度科幻；
- 不受控文字。

## 7. 禁用项

- 过度低幼；
- 钱币、金条、暴富符号堆叠；
- 复杂城市背景；
- 绿色金融图形大面积使用；
- 卡通人物表情过度夸张；
- 屏幕里出现不可控错误文字；
- 用图形暗示收益确定性。

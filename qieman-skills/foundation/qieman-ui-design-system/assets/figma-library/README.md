## 使用建议

1. 原始 `.fig` 文件体积较大，建议由团队在内部资产库单独维护。
2. 本 Skill 使用 Figma 组件库作为结构参考，不直接依赖图片切图。
3. 所有组件生成必须优先继承 `qieman-design-token-system` 中的语义 Token。
4. 组件命名、状态命名、变体命名需与 Figma Library 保持一致。

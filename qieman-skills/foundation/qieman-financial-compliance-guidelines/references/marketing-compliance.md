# Marketing Compliance

## 1. 适用范围

本文件用于约束且慢所有金融营销内容，包括：

- App Banner
- H5 活动页
- 小红书封面 / 九宫格
- 私域社群海报
- 高净值活动页
- 节日活动页
- 基金 / 投顾服务推广页
- AI 小顾功能宣传

## 2. 基本原则

金融营销不是单纯转化工具，而是投资者教育和服务解释的一部分。

营销内容必须遵循：

```text
真实准确 > 信息完整 > 风险清晰 > 表达克制 > 行动明确
```

## 3. 可以强调什么

可以强调：

- 服务价值
- 投资理念
- 长期陪伴
- 专业分析
- 账户诊断
- 信息透明
- 投顾服务流程
- 工具效率
- 用户权益说明

示例：

```text
帮你看清账户配置是否仍适合当前目标
市场波动时，先理解风险来自哪里
用长期配置陪你穿越不同市场环境
```

## 4. 不应强调什么

禁止或高风险强调：

- 确定收益
- 保本保收益
- 低风险高收益
- 限定损失
- 未来业绩预测
- 一夜暴富
- 立刻抄底
- 错过就亏
- 人人适合
- 专属捷径

## 5. 营销 CTA 规范

### 推荐 CTA

```text
了解详情
查看服务说明
查看风险提示
看看是否适合我
了解配置方案
查看账户诊断
预约顾问沟通
```

### 谨慎使用 CTA

```text
立即上车
马上抄底
锁定收益
立即抢购
最后机会
财富翻倍
```

## 6. 稀缺感表达规范

可以表达运营规则，但不能制造过度紧迫感。

### 可以

```text
本活动名额有限，具体以页面展示为准。
本服务需满足适当性要求后开通。
当前为阶段性开放体验，开通前请阅读服务说明和风险提示。
```

### 避免

```text
错过今天就没有了
最后上车机会
再不买就来不及了
名额正在疯抢
```

## 7. 高净值营销规范

高净值营销应表达“服务匹配”和“深度交流”，而不是制造财富优越感。

### 推荐方向

```text
为更复杂的资产需求，提供更深入的交流和陪伴。
面向满足适当性要求的用户开放专属服务说明。
与同频投资者交流配置思路，而非追逐短期热点。
```

### 避免方向

```text
300 万以上才配进的圈子
高净值专属赚钱机会
富人都在买
内部收益机会
财富阶层捷径
```

## 8. 设计层合规要求

- 风险提示不能使用过小字号、低对比度或隐藏在折叠区域。
- 收益、权益、限额信息不能明显大于风险说明太多。
- 不使用爆炸贴、强倒计时、红色抢购风格。
- 不用金币堆、钞票雨、暴富、豪车、豪宅等强刺激图形。
- 不用“稳赚”“躺赚”“暴富”等视觉字样。

## 9. 营销物料送审清单

- [ ] 是否存在收益承诺或保本暗示？
- [ ] 是否存在过度紧迫感？
- [ ] 是否存在不适当的高净值优越感表达？
- [ ] 是否标明活动规则、适用条件、有效期？
- [ ] 是否保留风险提示？
- [ ] 是否明确产品 / 服务主体？
- [ ] 是否避免将单一案例包装成普遍结果？

## 10. 监管依据摘要

基金销售机构和从业人员不得虚假记载、误导性陈述或重大遗漏；不得违规承诺收益、本金不受损失或限定损失金额、比例；不得预测基金投资业绩或宣传预期收益率。

基金宣传推介材料应加强投资者教育和长期投资理念，不应只追求短期营销转化。

> 本文件为设计生成约束，最终以现行法律法规、监管规定和公司合规审定为准。

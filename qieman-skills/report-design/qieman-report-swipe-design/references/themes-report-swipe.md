# 且慢报告翻页 Deck · 主题预设

**禁止自定义 hex**。只能从下列 3 套且慢报告主题选一套，整体替换 `template-report-swipe.html` 的 `:root` 块。

## 1 · 经典报告蓝（默认）

```css
:root {
  --brand: #1B88EE;
  --brand-deep: #1875D1;
  --brand-soft: #D8EBFF;
  --brand-pale: #EFF7FF;
  --bg-page: #E3EFFE;
  --bg-card: #FFFFFF;
  --bg-toc: #FFFFFF;
  --summary-bg: #9CCBF8;
}
```

适合：家庭财富报告、投顾报告、资产配置复盘。

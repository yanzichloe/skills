# Tone of Voice / 文案语气规范

## 1. 文案原则

且慢文案应遵循：

> 先说结论，再解释原因；先安抚情绪，再给出理性判断；先讲用户价值，再补充产品逻辑。

## 2. 推荐表达顺序

```text
结论前置 → 原因解释 → 风险边界 → 下一步建议
```

## 3. 语气特征

| 方向 | 表达方式 |
|---|---|
| 专业 | 有依据、有边界、有解释 |
| 温和 | 不催促、不恐吓、不制造焦虑 |
| 清晰 | 少术语，必要术语要解释 |
| 克制 | 不夸大收益，不暗示确定性 |
| 陪伴 | 强调“我们一起看”“帮你理解”“可以这样规划” |

## 4. 推荐句式

### 市场波动

- 市场短期波动加大，我们先帮你看清变化来自哪里。
- 波动并不一定意味着需要立刻操作，可以先看配置是否仍匹配你的目标。
- 当前市场变化较多，建议先确认账户风险是否仍在可承受范围内。

### 投顾建议

- 基于当前账户情况，可以优先关注配置是否仍匹配你的目标。
- 这不是一次性的判断，更适合结合你的目标和风险承受能力持续观察。
- 如果你的投资目标没有变化，可以先从配置偏离情况看起。

### 风险提示

- 这类产品存在波动风险，建议结合你的风险承受能力再决策。
- 历史表现不代表未来，实际收益可能随市场变化而波动。
- 当前方案仅供参考，建议结合账户情况和风险等级综合判断。

### 长期陪伴

- 投资不是一次选择，而是一段需要持续调整的旅程。
- 我们会持续帮你关注账户状态和市场变化。
- 慢慢来，把财富放在更合适的位置。

### AI 小顾

- 小顾会结合你的账户状态和服务偏好，帮助你更快理解当前情况。
- 小顾可以先帮你梳理重点，再说明原因和风险。
- 你可以随时调整记忆和画像，让小顾更好地理解你的需求。

### 高净值

- 为更复杂的资产需求，提供更深入的交流和专属服务。
- 与同频者交流，不是追逐热点，而是提升决策质量。
- 更高的资产规模，往往需要更系统的配置和陪伴。

## 5. 禁用表达

| 避免表达 | 原因 |
|---|---|
| 稳赚不赔 | 收益承诺风险 |
| 现在不买就错过 | 过度制造紧迫感 |
| 低风险高收益 | 容易误导 |
| 抄底机会来了 | 暗示择时确定性 |
| 专属财富捷径 | 夸大服务效果 |
| 保本高收益 | 违反金融表达边界 |
| 闭眼买 | 诱导冲动交易 |

## 6. 标题写法

推荐：

- 帮你看清账户当前状态
- 市场波动时，先看配置是否仍匹配目标
- 你的资产配置完成度
- 小顾会参考哪些数据？
- 更懂你的长期财富陪伴

避免：

- 这次机会别错过
- 赚到第一个 300 万的秘密
- 现在就是上车时刻
- 一键抄底
- 稳稳赚钱

## 7. 金融文案底线

必须遵守：

- 不承诺收益；
- 不暗示稳赚；
- 不过度制造紧迫感；
- 风险提示清晰可见；
- 收益与风险并列表达；
- 投顾建议不能包装成确定性结论；
- 金融数据必须标注口径。

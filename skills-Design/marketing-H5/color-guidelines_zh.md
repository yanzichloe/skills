# 且慢营销配色规范

在且慢 UI 品牌主色基础上，扩展品牌蓝 A、稳重深蓝 B、科技紫 C、新春红金、高净值鎏金蓝，统一营销场景色彩风格。**主按钮/CTA 固定品牌蓝，不随基调改变。**

## When to Use

- 设计或实现且慢 H5、落地页、运营位、战报、数据长图
- 高净值/私募页、AI/创新功能宣发
- 需选择背景、标题、装饰色时
- 所有场景下主 CTA 均为品牌蓝按钮

## Instructions

### 1. 选色决策流程

1. **选基调**：日常/品牌→A；深度/高端→B；AI/创新→C；春节→红金；高净值→B+金。
2. **定层级**：600 主视觉点睛、500 hover、100–200 背景氛围。
3. **主 CTA**：**固定品牌蓝 `#1B88EE`**，不随基调改变。
4. **通用降噪**：用 Neutral + Solid 做底色与正文。

### 2. 主 CTA 强制约定

无论使用哪套基调，**主按钮 / 立即参与 / 入金等 CTA 一律使用品牌蓝**：

| 元素 | 规范 |
|------|------|
| 主按钮 | 背景 `#1B88EE`，文字 `#FFFFFF`，圆角 8px，内边距 12px 24px |
| 次按钮 | 文字 `#1B88EE` 或轻底 `rgba(27,136,238,0.1)` |
| 禁用 | 背景/文字 `#CCCCCC` |

**口诀**：基调管背景和氛围，**按钮永远品牌蓝**。

### 3. 基调与场景

| 基调 | 场景 | 主色用途 |
|------|------|----------|
| A 品牌蓝 | 常规活动、通用 | 主 KV、CTA、品牌识别 |
| B 稳重深蓝 | 投研、高端、策略 | 深色 KV、数据模块 |
| C 科技紫 | AI、创新 | 背景、标题、装饰（非按钮） |
| 新春红金 | 春节、节庆 | 背景、氛围、装饰（非按钮） |
| B+金 | 高净值、私募 | 深蓝底 + 金色点睛 |

## Component Reference

### A 品牌蓝（100–1000）

| 等级 | HEX | 用途 |
|------|-----|------|
| 100 | `#F5F9FF` | 超浅背景、整页铺底 |
| 200 | `#DBEBFF` | 摘要卡、轻高亮 |
| 300 | `#9BCBF8` | 描边、图标 |
| 400 | `#67B0F4` | 渐变高亮 |
| 600 | `#1B88EE` | **品牌主色、主按钮** |
| 700 | `#125EA5` | 深色背景文字 |
| 1000 | `#061D32` | 近黑蓝 |

**渐变**：`#1B88EE → #67B0F4`（首屏）；`#0A3257 → #1B88EE`（深色模块）

### B 稳重深蓝

| 等级 | HEX | 用途 |
|------|-----|------|
| 100 | `#F0F6FF` | 浅底、信息块 |
| 700 | `#125EA5` | 页头深蓝 |
| 1000 | `#061D32` | 近黑深蓝 |

**渐变**：`#061D32 → #125EA5`（高净值首屏）

### C 科技紫

| 等级 | HEX | 用途 |
|------|-----|------|
| 100 | `#F5F0FF` | 浅紫背景 |
| 200 | `#E7DBFF` | 卡片淡紫底 |
| 600 | `#7477D8` | 主视觉（非按钮） |

**渐变**：`#7477D8 → #BB99FF`（AI 首屏）

### 新春红金

| 色系 | 100 | 200 | 600 |
|------|-----|-----|-----|
| 红 | `#FEEBE7` | `#FDDBD3` | `#FA420A` |
| 金 | `#FFF7E5` | `#FFEECC` | `#FFAA00` |

**渐变**：`#FA420A → #F88D72`（首屏）；`#FFAA00 → #FFDD99`（高光）

### Neutral + Solid

| 名称 | HEX | 用途 |
|------|-----|------|
| Neutral 100 | `#FCFDFD` | 通用浅底 |
| Neutral 200 | `#F2F5F7` | 内容区底色 |
| Neutral 400 | `#C3CBD5` | 分割线、图表辅助 |
| Solid White | `#FFFFFF` | 白底、卡片 |
| Solid 900 | `#363636` | 正文 |
| Solid 1000 | `#1F1F1F` | 标题 |

## Examples

- **常规活动 H5**：A 基调，背景 `#F5F9FF`，主按钮 `#1B88EE`
- **新春红包**：红金背景 `#FEEBE7` / `#FFF7E5`，主按钮仍 `#1B88EE`
- **投研报告**：B 基调，深蓝 KV `#061D32 → #125EA5`，CTA `#1B88EE`
- **AI 功能**：C 基调，紫底 `#F5F0FF`，CTA `#1B88EE`
- **高净值**：B 深蓝底 + Gold 400–600 小面积描边，CTA `#1B88EE`

---

## Guidelines

### 使用口诀

- **选主题**：日常→A，深度→B，AI→C，春节→红金，高净值→B+金
- **主色档位**：主视觉 600；主按钮固定 `#1B88EE`
- **控数量**：一页一个主色基调；红/金/紫不用于主 CTA
- **先清晰再好看**：优先保证可读性与层级

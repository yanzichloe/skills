# Risk Disclosure

## 1. 适用范围

本文件用于规范且慢所有页面和物料中的风险披露，包括：

- 产品详情页
- 策略详情页
- 交易确认页
- 卖出 / 买入 / 调仓弹窗
- 财富报告
- 投顾服务说明
- 高净值营销页
- 私募专区
- AI 小顾回复
- H5 / 海报 / Banner

## 2. 风险提示原则

风险提示必须：

```text
醒目、明确、易读、靠近决策点、与收益展示并列
```

不能：

```text
隐藏、弱化、折叠、低对比、过小字号、仅放在页面底部
```

## 3. 风险提示层级

| 层级 | 使用场景 | 展示方式 |
|---|---|---|
| 一级风险 | 交易确认、私募、高风险产品 | 强提示卡片 / 弹窗 / 勾选确认 |
| 二级风险 | 策略页、产品介绍、收益展示 | 与收益信息同屏展示 |
| 三级风险 | 营销海报、Banner、入口卡片 | 底部短提示 + 跳转完整说明 |

## 4. 常用风险提示模板

### 基金产品

```text
基金有风险，投资需谨慎。产品过往业绩不代表未来表现，投资前请仔细阅读基金合同、招募说明书、产品资料概要等法律文件。
```

### 投顾服务

```text
投顾服务提供的是基金组合投资建议，不保证投资收益，也不保证本金不受损失。请结合自身风险承受能力审慎决策。
```

### 收益展示

```text
以上数据为历史表现或测算结果，不代表未来收益，也不构成收益承诺。
```

### AI 分析

```text
AI 分析基于当前可获取的信息生成，仅供辅助理解，不构成确定性投资建议或收益保证。
```

### 私募基金

```text
私募基金仅面向符合条件的合格投资者，具有较高风险和流动性限制，投资前请充分了解产品风险并完成适当性匹配。
```

## 5. 风险提示放置位置

| 页面类型 | 建议位置 |
|---|---|
| 产品详情页 | 收益 / 风险数据附近、底部完整风险说明 |
| 策略页 | 策略介绍后、业绩图表附近、交易入口附近 |
| 交易确认页 | CTA 上方、确认勾选旁、弹窗中部 |
| H5 活动页 | 首屏利益点附近、活动规则区、底部 |
| 海报 / Banner | 视觉底部或利益点下方，保证可读 |
| AI 回复 | 结论后或行动建议前 |

## 6. 设计规范

- 风险提示字号不得小到无法阅读。
- 文案颜色对比度需满足基本可读性。
- 不得将风险提示做成极浅灰、透明、隐藏折叠。
- 涉及交易的风险提示应靠近主按钮。
- 涉及收益的风险提示应靠近收益数字或图表。
- 高风险场景建议使用图标 + 文案 + 说明卡片。

## 7. 检查清单

- [ ] 是否出现收益、业绩或历史表现？
- [ ] 是否紧邻收益展示补充风险提示？
- [ ] 是否明确历史不代表未来？
- [ ] 是否提醒用户阅读法律文件？
- [ ] 是否提示适当性和风险承受能力？
- [ ] 是否在交易前给出明确风险说明？
- [ ] 是否避免把风险提示隐藏在最底部？

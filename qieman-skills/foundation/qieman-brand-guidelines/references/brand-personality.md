# Brand Personality / 品牌调性规范

## 1. 品牌角色

且慢是用户身边的长期财富陪伴者。

它的核心角色不是“交易推动者”，而是“买方投顾立场下的财富管理陪伴者”。

## 2. 一句话定位

> 且慢，是一个用专业投顾和长期陪伴，帮助用户更从容地管理财富的理财平台。

## 3. 品牌人格

且慢应像这样的人：

- 专业，但不强势；
- 理性，但有温度；
- 懂投资，也懂用户情绪；
- 能解释复杂问题，但不制造压迫感；
- 会提醒风险，也会给出清晰下一步。

## 4. 核心调性

| 调性 | 表达方式 | 避免方向 |
|---|---|---|
| 专业可信 | 数据清晰、逻辑严谨、风险说明充分 | 夸张营销、情绪煽动 |
| 温和克制 | 少用强刺激视觉，强调陪伴感 | 焦虑感、压迫感、倒计时促销感 |
| 长期主义 | 强调规划、配置、复利、周期 | 短期暴富、追热点、稳赚暗示 |
| 年轻清爽 | 浅色、留白、轻图形、友好表达 | 传统金融厚重、沉闷、复杂 |
| 有温度 | 用陪伴、理解、解释降低用户压力 | 冷冰冰的数据堆叠 |

## 5. 品牌关键词

```text
专业可信
温和克制
长期陪伴
年轻清爽
理性投资
资产配置
风险理解
安放财富
静待花开
```

## 6. 用户感受目标

用户看到且慢的页面、海报、报告或文案时，应产生以下感受：

- 这个平台是专业的；
- 它不是在催我交易；
- 它能帮我理解复杂的金融问题；
- 它关注长期规划，而不是短期刺激；
- 它让我更安心、更从容。

## 7. 品牌判断标准

一个设计是否符合且慢品牌，可以用以下问题判断：

- 是否帮助用户理解，而不是推动用户冲动操作？
- 是否表达了长期主义，而不是短期机会？
- 是否专业清晰，而不是堆砌金融术语？
- 是否有温度，但不低幼？
- 是否克制可信，而不是营销感过强？

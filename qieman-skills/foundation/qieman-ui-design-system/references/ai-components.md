# AI Components AI 专属组件

## 1. 定义

AI 专属组件用于 AI 小顾、对话记忆、且慢画像、主动提醒、账户分析、市场分析、投资规划解释等场景。

## 2. AI 组件类型

| 组件 | 用途 |
|---|---|
| AI Chat Bubble | AI 对话回复 |
| User Chat Bubble | 用户消息 |
| AI Insight Card | 账户 / 市场 / 策略分析结论 |
| Memory Card | 对话记忆展示与编辑 |
| Profile Card | 且慢画像 / 账户画像 |
| Source Card | 数据来源说明 |
| Task Card | 主动任务、提醒、监控 |
| Risk Boundary Notice | AI 建议边界提示 |
| Suggested Actions | 后续可操作建议 |

## 3. AI 回复结构

金融 AI 回复建议采用：

```text
[先说结论]
[为什么]
[和你账户有什么关系]
[风险或不确定性]
[你可以怎么做]
```

## 4. AI 分析卡片 Anatomy

```text
[AI 标识 / 小顾头像]
[结论标题]
[关键洞察]
[数据来源]
[风险边界]
[建议操作]
```

## 5. 且慢画像组件

画像组件必须强调用户控制权：

- 展示数据来源；
- 支持开启 / 关闭；
- 支持查看、编辑或清除；
- 说明更新频率；
- 不展示过度敏感推断。

推荐结构：

```text
且慢画像
小顾会参考你的账户状态、投资规划和服务偏好，帮助理解你的需求。
[生成且慢画像]
```

## 6. 记忆组件

记忆卡片用于展示长期偏好，不应像标签墙一样堆满。

分类建议：

- 投资目标；
- 操作偏好；
- 关注方向；
- 沟通偏好；
- 风险舒适区。

## 7. 视觉规则

| 元素 | 规则 |
|---|---|
| AI 主色 | 品牌蓝 + 浅紫辅助 |
| 背景 | 轻蓝 / 白色 / 透明卡片 |
| 卡片 | 圆角 16px，轻阴影 |
| 图标 | 简洁、科技感、不过度拟人 |
| 动效 | 温和，避免过强炫技 |

## 8. AI 合规边界

AI 组件必须避免：

- 直接承诺收益；
- 输出确定性买卖指令；
- 暗示模型能预测市场；
- 隐藏数据来源；
- 用“最优”“稳赚”“必涨”等表述。

推荐提示：

```text
以上分析基于当前账户和平台数据生成，仅供参考，不构成收益承诺。市场有波动，请结合风险承受能力审慎决策。
```

## 9. Suggested Actions

建议操作应控制数量，优先 1-3 个。

示例：

```text
[查看配置偏离]
[重新测评风险]
[咨询顾问]
```

## 10. 禁用

- 不让 AI 组件显得像强制投资建议；
- 不把 AI 结论置于风险提示之上且无边界说明；
- 不用过度卡通化 IP 弱化金融专业感；
- 不显示不可控错误文字。

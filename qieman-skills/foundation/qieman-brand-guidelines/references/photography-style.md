# Photography Style / 摄影素材风格规范

## 1. 摄影总原则

且慢摄影风格应避免传统金融的“西装商务 + 城市天际线 + 强销售感”，更接近：

```text
自然
生活化
低饱和
长期陪伴
从容规划
```

## 2. 推荐摄影方向

| 方向 | 画面特征 | 适用场景 |
|---|---|---|
| 自然留白 | 天空、海面、山丘、晨光、窗边 | 品牌主 KV、长期投资主题 |
| 生活理财 | 家庭、书桌、阅读、沟通、规划 | 投顾服务、养老、家庭资产 |
| 轻商务 | 顾问沟通、办公协作、专业服务 | 顾问介绍、机构合作 |
| 温和科技 | 柔光屏幕、数据光效、透明界面 | AI 小顾、智能投顾、开放平台 |
| 高净值 | 低饱和空间、质感材质、局部特写 | 私域权益、高净值圈层 |

## 3. 摄影关键词

- 清晨光感；
- 柔和自然光；
- 低饱和；
- 大留白；
- 空气感；
- 安静、稳定、从容；
- 不过度摆拍；
- 避免强销售感。

## 4. 人物摄影

人物应表达：

- 自然；
- 可信；
- 平和；
- 有专业感；
- 适度亲和。

避免：

- 夸张表情；
- 过度摆拍；
- 传统商务假笑；
- 过度成功学姿态；
- 明显炫富场景。

## 5. 场景摄影

推荐场景：

- 书桌、笔记本、电脑、规划资料；
- 家庭沟通、顾问交流；
- 清晨窗边、安静室内；
- 海边、山丘、天空、自然远景；
- 柔和科技屏幕和数据光效。

避免场景：

- 城市 CBD 滥用；
- 股票大厅；
- 夸张交易屏幕；
- 金库、金条、钞票堆；
- 豪车、豪宅、奢侈品符号。

## 6. 构图建议

- 画面保持充足留白；
- 主体不宜过满；
- 适合承载标题和品牌信息；
- 优先横向或竖向可裁切素材；
- 色温偏柔和，避免强对比硬光。

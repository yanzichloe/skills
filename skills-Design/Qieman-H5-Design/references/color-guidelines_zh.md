# 且慢营销 H5 配色规范

## Purpose
本文件用于定义且慢营销 H5 在颜色使用上的通用规则，帮助在活动页、专题页、年度账单页、节日页等场景中统一主色、辅助色、背景色、文本色与状态色，并明确主 CTA 与场景色板的使用方式。

本规范重点解决以下问题：

- 品牌蓝与多场景基调的配合
- 主 CTA 固定品牌蓝、不随页面基调改变
- 页面背景与层级用色（浅色、轻盈、清晰）
- 颜色用于建立信息层级，而非单纯装饰
- HTML Demo / 页面原型中的颜色约束

## When to read
在以下情况应优先读取本文件：

- 需要为营销 H5 选择背景色、标题色、卡片色
- 需要确定主按钮、次按钮、链接、强调色
- 需要按场景（常规、节日、科技、高净值）选用色板
- 需要输出或实现 HTML Demo 中的 CSS 变量与色值
- 需要与 header-hero、card-styles 等 reference 配合保证视觉一致

## Key rules
- 品牌蓝为默认主色；主 CTA 固定 `#1B88EE`，不随基调改变。
- 页面背景优先浅色、轻盈、清晰；用颜色建立层级，不追求热闹。
- 一页一个主色基调；红/金/紫可用于背景与装饰，不用于主 CTA。
- 优先保证可读性与层级，再考虑氛围与装饰。

## Color system

### 主色
| 用途 | 色值 | 说明 |
|------|------|------|
| 品牌主色 / 主 CTA | `#1B88EE` | 主按钮、分区竖条、重点强调，不随场景改变 |
| 主色浅档（背景/高亮） | `#F5F9FF`（100）、`#DBEBFF`（200） | 整页铺底、摘要卡、轻高亮 |
| 主色中档（描边/图标） | `#9BCBF8`（300）、`#67B0F4`（400） | 描边、图标、渐变高亮 |
| 主色深档（深色区文字） | `#125EA5`（700）、`#061D32`（1000） | 深色背景上的文字、近黑蓝 |

### 辅助色
| 用途 | 色值 | 说明 |
|------|------|------|
| 稳重深蓝 | `#F0F6FF`、`#125EA5`、`#061D32` | 投研、高端、深色 KV |
| 科技紫 | `#F5F0FF`、`#E7DBFF`、`#7477D8` | AI、创新场景背景与装饰（非按钮） |
| 新春红 | `#FEEBE7`、`#FDDBD3`、`#FA420A` | 春节背景、氛围、装饰（非按钮） |
| 金 | `#FFF7E5`、`#FFEECC`、`#FFAA00` | 节日高光、高净值点睛（非按钮） |

### 背景色
| 用途 | 色值 |
|------|------|
| 页面底 | `#F9FAFB` |
| 通用浅底 | `#FCFDFD`（Neutral 100） |
| 内容区底色 | `#F2F5F7`（Neutral 200） |
| 卡片 / 白底 | `#FFFFFF` |
| 摘要/高亮块 | `#DBEBFF`、`#FFF7E5`（数据卡黄底） |

### 文本色
| 用途 | 色值 |
|------|------|
| 标题 | `#333333` / `#1F1F1F`（Solid 1000） |
| 正文 | `#606060` / `#363636`（Solid 900） |
| 辅助 / 说明 | `#999999` |
| 弱提示 / 禁用 | `#CCCCCC` |
| 深色底上的正文 | `#FFFFFF` 或半透明白 |

### 状态色
| 用途 | 色值 |
|------|------|
| 成功 / 正收益 | `#07AD8F`、`#FF6633`（正收益强调可用橙红） |
| 负收益 | `#07AD8F`（绿） |
| 提醒 | `#EA9500` |
| 警示 / 错误 | `#FA420A`、`#FA440C` |
| 分割线 / 图表辅助 | `#C3CBD5`（Neutral 400）、`#D8D8D8` |

## Scenario palettes

### 常规营销场景
- 基调：品牌蓝 A。
- 背景：`#F5F9FF`、`#DBEBFF`；卡片 `#FFFFFF`；页面底 `#F9FAFB`。
- 主视觉与 CTA：`#1B88EE`；渐变可用 `#1B88EE → #67B0F4`。
- 适用：通用活动页、年度账单、数据总结、品牌传播。

### 节日场景（如春节）
- 基调：新春红金。
- 背景：`#FEEBE7`、`#FFF7E5`、`#FDDBD3`；装饰可用 `#FA420A`、`#FFAA00`。
- 主 CTA：仍为 `#1B88EE`，不改为红或金。
- 适用：春节、节庆专题；氛围用红金，按钮保持品牌蓝。

### 科技感场景（如 AI 专题）
- 基调：科技紫 C。
- 背景：`#F5F0FF`、`#E7DBFF`；主视觉点缀 `#7477D8`（非按钮）。
- 主 CTA：`#1B88EE`。
- 渐变：`#7477D8 → #BB99FF` 可用于首屏背景或装饰。
- 适用：AI 功能、智能投顾、创新功能宣发。

### 高净值感场景
- 基调：稳重深蓝 B + 金点缀。
- 背景：深色 Hero 可用 `#061D32 → #125EA5`；浅区 `#F0F6FF`。
- 点睛：金 `#FFCC66`、`#FFAA00` 小面积描边或图标，不用于主 CTA。
- 主 CTA：`#1B88EE`。
- 适用：投研、高端、私募、策略类页面。

## CTA color rules
- **主按钮**：背景 `#1B88EE`，文字 `#FFFFFF`；圆角 8px 或胶囊形 9999px；内边距 12px 24px；无边框、无投影。所有场景一致。
- **次按钮**：文字 `#1B88EE`，或背景 `rgba(27,136,238,0.1)`，边框可选 `1px solid #9BCBF8`；圆角与主按钮统一。
- **链接**：文字 `#1B88EE`；hover 可用 700 档深蓝或下划线。
- **强调色**：正文内强调用 `#1B88EE`；正收益可用 `#FF6633`，负收益用 `#07AD8F`；警示用 `#FA420A`。
- **禁用**：背景或文字 `#CCCCCC`。

主 CTA 不随页面基调改变；红/金/紫仅用于背景与装饰，不用于主按钮。

## Dos
- 主 CTA 统一使用品牌蓝 `#1B88EE`。
- 页面背景优先浅色（100–200 档），保证可读与层级。
- 一页一个主色基调；用 Neutral + Solid 做底色与正文降噪。
- 数据卡、摘要卡与状态色与本文本、card-styles 一致。
- HTML Demo 中可用 CSS 变量（如 `--qm-brand-primary`、`--mkt-a-100`）便于维护。

## Don'ts
- 不要将主 CTA 改为红、金、紫或其他基调色。
- 不要在一页混用多套主色基调导致杂乱。
- 不要用高饱和撞色、霓虹渐变破坏金融感。
- 不要忽略对比度；深色底须配浅字。

## Examples

**示例 1：年度账单页**  
背景 `#F5F9FF`、`#DBEBFF` 渐变；卡片 `#FFFFFF`；摘要卡浅蓝底 `#DBEBFF`，边框 `#9CCBF8`；正收益 `#FF6633`，负收益 `#07AD8F`；主 CTA「查看我的账单」「生成年度海报」均为 `#1B88EE`；数据来源与免责 `#999999`。

**示例 2：春节活动页**  
背景 `#FEEBE7` 或 `#FFF7E5`；装饰与顶条可用 `#FDDBD3`、`#FA420A`、`#FFAA00`；主 CTA「立即参与」仍为 `#1B88EE`；正文与标题层级同常规，不因节日改变按钮色。

**示例 3：AI 专题页**  
背景 `#F5F0FF`、`#E7DBFF`；首屏或卡片点缀 `#7477D8`；主 CTA「立即体验」`#1B88EE`；图标与装饰可用科技紫，按钮保持品牌蓝。

**示例 4：高端服务页**  
深色 Hero 背景 `#061D32 → #125EA5`，文字白；内容区浅底 `#F0F6FF`，卡片 `#FFFFFF`；金色 `#FFCC66` 小面积描边或图标；主 CTA `#1B88EE`；整体稳重、克制。

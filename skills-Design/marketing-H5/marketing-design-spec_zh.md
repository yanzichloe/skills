# 且慢营销设计规范

指导且慢品牌营销类视觉设计，延续 UI 设计系统，补齐营销场景的**信息层级、模块模板、出图规格**，提供**H5 模块模板库**与推荐拼装方案。

## When to Use

- 设计或实现且慢 H5、落地页、专题页、战报页
- 运营位：Banner、弹窗、卡片、引导浮层、站内信配图
- 投放素材：信息流、短视频封面、开屏/贴片静帧
- 需保持且慢品牌一致性的营销物料

## Instructions

### 1. 设计原则

- **一致**：与产品内 UI 视觉同源
- **清晰**：信息可读、层级明确
- **可信**：金融属性、理性克制，避免强促销误导
- **可复用**：固定组件与版式，批量产出

### 2. 视觉基调（强制）

**关键词**：专业可信、克制清爽、数据友好、温和不压迫、理性但不冷。

**优先级**：信息清晰 > 层级明确 > 一致性 > 装饰点缀。

**禁止**：高饱和撞色、霓虹渐变、厚重投影、爆炸贴、粗描边大字、强烈红黄堆叠、情绪化拟人插画。

### 3. 色彩系统

| 用途 | 色值 |
|------|------|
| 主色/CTA | `#1B88EE` |
| 页头背景 | `#3180EC` |
| 重要标题 | `#333333` |
| 正文 | `#606060` |
| 辅助 | `#999999` |
| 弱提示 | `#CCCCCC` |
| 成功 | `#07AD8F` |
| 提醒 | `#EA9500` |
| 警示 | `#FA440C` |
| 页面底 | `#F9FAFB` |
| 卡片 | `#FFFFFF` |
| 信息淡底 | 主 `#F0F6FF` / 提醒 `#FFFAEF` / 警示 `#FEEDE9` |
| 边框 | 默认 `#D8D8D8`；蓝 `#9CCBF8` / 黄 `#FFDC9E` / 红 `#FAB6A5` |

### 4. 字体规范（强制）

**统一采用阿里巴巴普惠体（Alibaba PuHuiTi）**。字体文件放在 `且慢营销规范/fonts/`。

| 字重 | font-weight | 用途 |
|------|-------------|------|
| Light | 300 | 辅助说明 |
| Regular | 400 | 正文、副标题 |
| Medium | 500 | 小标题、强调 |
| Bold | 600–700 | 主标题、分区标题 |
| Heavy | 800 | 超大数字、主标题 |

备选：PingFang SC、微软雅黑（投放工具不便嵌入时）。

### 5. 排版规格

| 层级 | 字号 |
|------|------|
| 主标题 | 32pt（首屏）/ 26pt（头图） |
| 分区标题 | 26pt / 24pt / 19pt |
| 正文 | 16pt |
| 说明 | 14pt / 12pt |

营销头图：主标题 26pt 加粗、副标题 16pt 常规、备注 14pt；左对齐，与氛围图顶对齐（`align-items: flex-start`）；主副标题间距 8px，副标题与备注 4px。

### 6. 间距与圆角

- 模块间距：16px
- 卡片内边距：12px
- 圆角：8px / 12px / 24px（少量）

### 7. Logo 选用

| 背景 | 目录 | 说明 |
|------|------|------|
| 浅色 | `且慢营销规范/logo/浅色背景/` | 蓝标+蓝字，`且慢logo-regular.png` |
| 深色 | `且慢营销规范/logo/深色背景/` | 白标+白字，`且慢logo-white.png` |

同一模块内只使用一套 Logo，不混用。

### 8. 信息层级（建议顺序）

1. 标题（一句话利益点）
2. 关键利益点（最多 3 条）
3. 关键数据/证明
4. 行动引导（CTA）
5. 风险提示（小字号、次级色）

### 9. 主 CTA 规范

- 背景 `#1B88EE`，文字 `#FFFFFF`
- 胶囊形 `border-radius: 9999px`，内边距 12px 24px
- 无边框、无投影
- 一屏最多一个主 CTA

## Component Reference

### 首屏/页头

- 布局：左文右图，关键区域 4:3
- 背景：`#3180EC` 或浅色渐变（见《营销头部氛围图设计 SKILL》）
- 深色底用白字，浅色底见 3.2.1 排版

### 摘要卡

- 底色 `#DBEBFF`，边框 `1px solid #9CCBF8`，圆角 12px
- 结论（大字）> 数据（对齐）> 解释（小字）

### 内容卡

- 底色 `#FFFFFF`，圆角 8px，弱投影或无投影

### 风险/提示卡

- 提醒：底 `#FFFAEF` + 边 `#FFDC9E`
- 警示：底 `#FEEDE9` + 边 `#FAB6A5`

### 图标

- 使用 [Remix Icon](https://remixicon.cn)，CDN：`https://cdn.jsdelivr.net/npm/remixicon@4.2.0/fonts/remixicon.css`
- 线性优先（`-line` 系列），尺寸 16/20/24/32px
- 颜色仅三档：`#333333` / `#999999` / `#1B88EE`

## Examples

### H5 模块拼装

**强转化活动**：Hero（带 CTA）→ 卖点卡×2 → 流程卡 → 规则（折叠）→ CTA → 风险提示

**功能宣发**：Hero → 卖点卡×3 → 权益清单 → 对比/证明 → FAQ → CTA → 风险提示

**高净值/会员**：深色 Hero → 数字证据 → 权益清单 → 口碑 → FAQ → CTA → 风险提示

### 模块 01｜Hero

主标题（1 行）+ 副标题（1–2 行）+ 标签（可选）+ 主 CTA（可选）

### 模块 02｜卖点卡

插画/图标 → 标题（8–14 字）→ 要点 2–3 条 → 辅助说明

### 模块 03｜数字证据

数字（大）+ 单位（小）+ 解释（1 行）；三列布局

### 模块 04｜流程卡

模块标题 → 步骤 3–5 个，每步：标题 + 解释

### 模块 05–11

权益清单、对比、口碑、FAQ、规则、CTA、风险提示；详见 Appendix。

## Guidelines

### 数字与口径

- 单位统一（同页不混用元/万元/¥）
- 时间明确（如「截至 2026-01-29」）
- 对比对象写清

### 规格与交付

- 命名：`渠道_活动名_模块_尺寸_版本_日期`
- 输出：PNG/JPG/WebP；HTML + CSS + 资源
- 移动端：关键文案与 CTA 在安全区内

### 质检清单

- [ ] 颜色/字号/圆角是否使用且慢体系
- [ ] 是否采用阿里普惠体
- [ ] 标题/利益点/数据/免责是否层级清晰
- [ ] 对比度足够，小字≥12pt
- [ ] CTA 明确，一屏不多主按钮
- [ ] 风险提示可见且口径正确
- [ ] 数据口径完整
- [ ] 装饰克制，能删就删

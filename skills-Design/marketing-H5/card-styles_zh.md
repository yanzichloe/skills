# 且慢 H5 卡片风格设计规范

设计 H5 营销页的卡片式布局、模块化板块、Hero 区及内容卡组件的视觉与结构。适用于互认基金、投研专题、年度账单等场景。

## When to Use

- 设计或实现且慢 H5 营销页（互认基金、投研专题、产品介绍、年度账单）
- 落地页、长图页中的卡片式内容区
- 需要「不同人物/插画 + 不同背景色」的 Hero 变体
- 需要数据可视化卡、柱状图、用户对比图、年度总结卡等组件

## Instructions

### 1. 页面结构

采用**三段式结构**：Hero 卡 → 内容卡 → 底部扩展区。

| 区域 | 用途 |
|------|------|
| **顶部 Hero 卡** | 主标题 + 简介 + 人物/概念插画；背景色随插画主色变化；投研/策略类可选深色 Hero（稳重深蓝 B 基调，白字） |
| **中部内容卡** | 投资目标、配置组合、类型说明等，多个独立卡片纵向堆叠 |
| **底部扩展区** | 免责声明、风险提示，小字号、次级色 |

### 2. 卡片视觉规范（强制）

| 属性 | 取值 |
|------|------|
| **背景** | `#FFFFFF`，页面底 `#F9FAFB` |
| **边框** | 无 |
| **阴影** | 无，以留白区分层次 |
| **圆角** | 8px / 12px |
| **内边距** | 12px～16px |
| **模块间距** | 16px |

### 3. 卡片形态选择

根据内容选择以下形态之一或组合：顶条标题卡、顶条+图标装饰卡、数据可视化卡、数据对比统计卡、柱状图卡、用户对比图卡、年度总结卡、风险警示区块、图标组卡、分区标题卡。详见下方 Component Reference。

### 4. 样式切换（强制）

卡片样式须支持**刷新后更换**。实现方式：URL 参数 `?card=<index>` 或 `localStorage`/`sessionStorage` 记录索引；根元素设置 `data-card-style="banner"` / `"section"` / `"banner-icon"`；建议支持 3 种形态轮换。

### 5. 主 CTA

- 背景 `#1B88EE`，文字 `#FFFFFF`
- 胶囊形 `border-radius: 9999px`，内边距 12px 24px
- 无边框、无投影

## Component Reference

### Hero 卡

| 属性 | 取值 |
|------|------|
| 背景 | 浅色渐变（100–200 档）或深色 Hero（`#061D32`～`#125EA5`） |
| 主标题 | 26pt 加粗，浅色底 `#333333`，深色底 `#FFFFFF` |
| 简介 | 16pt 常规，`#606060` 或浅色半透明 |

### 彩色顶条标题卡

| 属性 | 取值 |
|------|------|
| 顶条背景 | 浅蓝 `#DBEBFF` / 浅橙 `#FDDBD3` / 浅紫 `#E7DBFF` |
| 标题 | 16pt～19pt 加粗、居中 |

### 数据对比统计卡

- 浅黄底区块 `#FFF7E5`，细边框 `#FFDD99`
- 白底数据格：大数字 16–24pt 加粗，小标签 12–14pt
- 柱状图：深蓝 `#125EA5` 与金色 `#FFCC66` 交替

### 风险警示区块

- 背景 `#FEEDE9` / `#FEEBE7`，图标 `#FA420A`

### 年度总结卡

- 主指标居中，正收益橙红 `#FF6633`，负收益绿 `#07AD8F`
- 麦穗装饰：左右对称，高度≈主指标 60%～70%，素材 `且慢营销规范/麦穗/麦穗装饰-年度收益率.png`
- 白底数据格、成就标签（图标+次数）

### 用户对比图卡（VS 双柱）

- 左：浅灰圆柱 + 浅灰气泡 `#F5F5F5`
- 右：浅粉圆柱 `#FDDBD3` / `#FAB5A3` + 浅粉气泡
- VS 分隔：V 蓝 `#1B88EE`，S 红 `#FA420A`
- 底部结论句含倍数或情绪词

### 柱状图卡片

- 业绩走势：产品橙 `#FB5D2D`、基准蓝 `#1B88EE`、指数浅蓝 `#9BCBF8`
- 净值对比：产品红 `#FA420A`、基准灰 `#C3CBD5`
- 圆柱对比：品牌蓝 `#1B88EE`、橙 `#FB5D2D`
- **必附**数据来源与统计周期，12pt、`#999999`

### 分区标题

- 蓝竖条：4px×20px，`#1B88EE`
- 标题：19pt 加粗，`#333333`，与竖条间距 8px

## Examples

- 投资目标说明：顶条标题卡（浅蓝顶条）+ 正文 + 底部图位
- 配置组合展示：数据可视化卡（环形图 + 图例）+ 底部说明
- 收益/回撤对比：数据对比统计卡（浅黄底 + 白底数据格 + 柱状图）
- 清仓 vs 收益 Top10%：用户对比图卡（VS 双柱 + 气泡 + 结论句）
- 年度账单：年度总结卡（主指标 + 麦穗 + 数据格 + 成就标签）
- 风险事件提醒：风险警示区块（浅粉底 + 红色图标）

## Guidelines

### 实现要点

- 内容卡用 `data-module` 或 `data-removable` 支持快速隐藏
- 容器宽度：`min(343px, 100%)` 或 `min(430px, 100%)`，居中
- 人物/插画背景色按主色适配，与《营销头部氛围图设计 SKILL》一致

### 质检清单

- [ ] Hero 区主标题、简介、插画是否顶对齐？深色 Hero 是否白字？
- [ ] 内容卡是否白底、无边框、无投影、圆角 8/12px？
- [ ] 主 CTA 是否为品牌蓝 `#1B88EE`？
- [ ] 数据可视化图例颜色是否与环形图分段一致？
- [ ] 柱状图是否附数据来源与统计周期？
- [ ] 刷新是否有效切换卡片形态？（至少 2 种，建议 3 种）

### 与既有规范配合

| 规范 | 配合要点 |
|------|----------|
| 《且慢营销设计规范》 | 阿里普惠体、间距 16px、圆角 8/12px |
| 《且慢营销配色规范》 | 主按钮品牌蓝、背景 100–200 档 |
| 《营销头部氛围图设计 SKILL》 | Hero 区比例、背景色融合、`data-bg` |

## Appendix — 详细结构图与实现规格

### A. 页面布局示意

```
┌─────────────────────────────────┐
│  Hero 卡（主标题 + 简介 + 插画）  │
├─────────────────────────────────┤
│  内容卡 1（投资目标）             │
│  内容卡 2（配置组合）             │
│  内容卡 3（类型说明）             │
├─────────────────────────────────┤
│  底部：免责声明 / 风险提示        │
└─────────────────────────────────┘
```

### B. 顶条标题卡结构

```
┌─────────────────────────────────┐
│  ┌─────────────────────────┐    │  ← 彩色顶条
│  │ 标题文本文本             │    │
│  └─────────────────────────┘    │
│  正文说明（16pt）                │
│  底部图位                        │
└─────────────────────────────────┘
```

### C. 数据可视化卡（无顶条）

```
┌─────────────────────────────────┐
│ 标题（白底左对齐，16–19pt）       │
├─────────────────────────────────┤
│ 环形图  │  ● 60% 股票 描述...   │
│         │  ● 30% 债券 描述...   │
│         │  ● 10% 黄金 描述...   │
├─────────────────────────────────┤
│ 【产品名】采用...配置...         │
└─────────────────────────────────┘
```

### D. 柱状图子类型

| 类型 | 柱色 | 用途 |
|------|------|------|
| 业绩走势 | 橙 `#FB5D2D`、蓝 `#1B88EE`、浅蓝 `#9BCBF8` | 产品 vs 基准 vs 指数 |
| 净值对比 | 红 `#FA420A`、灰 `#C3CBD5` | 近一年/近三年排名 |
| 圆柱对比 | 蓝 `#1B88EE`、橙 `#FB5D2D` | 2 项简单对比 |
| 方案推荐 | 蓝/橙/灰 横向堆叠条 | 风险偏好推荐 |
| 大类资产 | 橙 `#FB5D2D`、灰 `#C3CBD5` | 分组柱状图 |
| 折线图 | 蓝 `#1B88EE` | 价格/净值曲线 |

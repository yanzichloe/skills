# Typography / 文字排版规范

## 1. 排版原则

且慢文字排版应体现：

```text
清晰
稳定
克制
易读
专业
```

排版目标不是炫技，而是帮助用户快速理解金融信息。

## 2. 字体气质

推荐字体气质：

- 现代；
- 清晰；
- 中性；
- 友好；
- 不厚重；
- 不花哨。

中文字体可优先选择系统默认无衬线字体或思源黑体类字体。

## 3. 层级建议

### App / 移动端

| 层级 | 建议字号 | 用途 |
|---|---:|---|
| 页面标题 | 20px - 24px | 页面主标题、弹窗标题 |
| 模块标题 | 16px - 18px | 卡片标题、分组标题 |
| 正文 | 14px - 16px | 正文说明、表单内容 |
| 辅助信息 | 11px - 13px | 时间、备注、风险说明 |
| 按钮文字 | 16px - 17px | 主操作按钮 |

### H5 / 海报

| 层级 | 建议方向 |
|---|---|
| 主标题 | 大而简洁，最多两行 |
| 副标题 | 解释利益点，不堆叠 |
| 内容条目 | 分组、短句化 |
| 规则说明 | 透明、可读，不隐藏 |

### Web / 平台页

| 层级 | 建议方向 |
|---|---|
| Hero 标题 | 稳定、大气、短句 |
| 模块标题 | 明确能力或价值 |
| 卡片标题 | 直给，不抽象 |
| 数据数字 | 对齐、清楚、口径可见 |

## 4. 数字排版

金融数字需要更高可读性：

- 数字与单位分离；
- 金额、比例、时间统一格式；
- 重要数字可加粗，但不要过度放大；
- 收益数据必须结合风险或口径说明；
- 不使用过度刺激的颜色强化收益。

## 5. 文本布局

推荐：

- 短句；
- 分组；
- 卡片化；
- 关键结论前置；
- 风险说明不折叠到不可见位置。

避免：

- 大段堆叠；
- 多层嵌套说明；
- 过度使用感叹号；
- 标题党；
- 口号化空话。

## 6. 标题风格

标题应清晰而克制。

推荐：

- “帮你看清账户当前状态”
- “市场波动时，先看配置是否仍匹配目标”
- “你的资产配置完成度”

避免：

- “稳赚机会来了”
- “错过再等一年”
- “马上抄底”
- “财富暴涨秘诀”

# Investment Expression

## 1. 定位

本文件用于规范且慢产品、运营、顾问和 AI 场景中的投资表达方式。

核心目标：

```text
把“确定性指令”改为“有条件的分析建议”
把“收益导向”改为“风险收益并列”
把“立即交易”改为“理解后决策”
```

## 2. 话术基本结构

推荐使用四段式表达：

```text
结论：当前可以关注什么
原因：为什么这样判断
风险：可能出现什么不利情况
行动：用户下一步可以怎么做
```

示例：

```text
从当前账户配置看，可以优先关注权益资产波动对整体回撤的影响。
主要原因是近期市场波动加大，组合中权益资产占比相对更高。
需要注意的是，短期市场方向具有不确定性，历史表现不代表未来。
你可以先查看账户诊断，再结合自己的风险承受能力决定是否调整。
```

## 3. 投资建议表达等级

| 等级 | 表达方式 | 使用场景 |
|---|---|---|
| 低风险 | “可以了解”“可以关注”“建议先查看” | 信息介绍、教育内容 |
| 中风险 | “可结合风险承受能力考虑”“如目标未变，可继续观察” | 账户分析、配置建议 |
| 高风险 | “不建议直接操作”“需完成适当性评估后再判断” | 交易、私募、高波动产品 |

## 4. 推荐表达

```text
可以关注
建议先了解
可结合你的风险承受能力判断
当前更适合作为长期配置的一部分理解
从资产配置角度看
这并不代表短期一定上涨
历史表现不代表未来收益
请结合产品风险等级和自身风险承受能力
```

## 5. 高风险表达

避免使用：

```text
建议立即买入
现在就是底部
一定会反弹
稳赚不赔
可以放心上车
风险很低收益很高
不用管波动
跟着买就行
```

## 6. 买入 / 卖出场景话术

### 买入

推荐：

```text
你可以先确认该产品风险等级是否与你的风险承受能力匹配。
如你认可该策略的长期投资目标，可以进一步查看交易规则和风险提示。
```

避免：

```text
现在买最合适
再不买就错过
低位上车稳赚
```

### 卖出

推荐：

```text
卖出前建议先确认这次操作是否会影响你的长期配置目标。
如果是因为短期波动产生焦虑，可以先查看账户回撤和策略说明。
```

避免：

```text
不要卖，后面肯定会涨
现在卖一定亏
坚持就一定赚钱
```

### 调仓

推荐：

```text
本次调整主要用于让组合配置更接近目标风险水平。
调仓不代表对短期市场方向的确定判断。
```

避免：

```text
调仓后收益会更好
这次能规避下跌
```

## 7. 顾问 / AI 场景表达

AI 或顾问辅助内容必须体现：

- 数据来源
- 分析边界
- 不确定性
- 用户适当性
- 风险提示

推荐：

```text
基于你在且慢平台的账户状态和服务偏好，小顾可以帮助你理解当前配置情况。
以下分析仅供参考，不构成对未来收益的保证。
```

## 8. 投资话术检查清单

- [ ] 是否把建议写成了确定性命令？
- [ ] 是否省略了风险和适当性条件？
- [ ] 是否暗示短期择时确定性？
- [ ] 是否过度强调收益而忽略回撤？
- [ ] 是否把历史表现包装成未来结果？
